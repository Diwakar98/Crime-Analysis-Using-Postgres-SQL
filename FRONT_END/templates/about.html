This page is about crime analysis in India across various regions.


While looking for datasets on the internet, we came across a dataset which had information about all the crimes that took place in India between 2001 and 2010. We thought of making a website where various attributes like year, state, crime category etc can be controlled by the user to get interactive analysis of the crime data. Admins like police officers etc could add data to the database if needed.

This website could be the one stop solution for someone who wanted to deeply analyse the crime data and write articles about various intricacies in the system. e.g. - One can analyse how the number of rape cases have increased or decreased in a particular area, how the number of confessions for a particular crime changed over the decade, how do the murder cases of male and female victims vary in a particulat state, and a lot more can be inferred by this website. The system has various crimes which a user can look into and analyse.

We really liked it because the data was looking real, was self explanatory and would only take a little effort to clean as most of the cells were not null.

The Landing Page contains - Different categories of Crime analysis, e.g. - crime against women, automobile theft, trial of violent crimes in courts, etc. Signup - Users can sign up only using a specific password in order to gain access to add new data into the database. Login - Users successfully signed up can add new data fields into the specified table after providing the correct username and password.

The webpages for different crime categories contain dropdown menus where you can select parameters like state, crime category, year, etc., which runs an sql query command in the backend, based on the input parameters. The result is then displayed in different tables and graphs for user-friendly comparison and analysis of data.

E.g.- 1. selecting Delhi, Additional Session Judge, District Session Judge in ‘Period of Trial by Courts’ displays graphs and tables of year-wise and sub-category-wise Number of cases that took less than a year, 1-5 years, 5-10 years, ….. In court. 2. Under ‘Automobile-theft’ category, user can analyse the number of automobiles traced, automobiles recovered, etc. for different types of automobiles like cars, buses, goods carrying vehicles, etc over the years 2000-2010.
